# LS-8 Assembler

This takes LS-8 assembler source and converts it into `.ls8` "binary"
files.

## Usage

This will produce a `.ls8` file for a given source.

```
node asm source.asm
```

## Features

* Labels
* String constants
* Numeric constants
* Comments
